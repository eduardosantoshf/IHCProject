# IHCProject

For this course's final project, we choose to develop an app that includes many features from top music streaming and music sharing apps like **Spotify**, **Apple Music/iTunes** and **SoundCloud**.

The other objective was to create a **social network** for users to develop their capabilities and improve their performance.

## Course
This project was developed under the [Human-computer Interaction](https://www.ua.pt/en/uc/2484) course of [University of Aveiro](https://www.ua.pt/).

## Requirements
Non-functional requirements:
* **Menu ‘help and documentation’**
* **Using global symbols, for example '+' to add**
* **Easy to use**
* **Desktop app for windows**

Functional Requirements/ App Features:
* **Play music**
* **Follow artist**
* **Like/Review**
* **Playlists**
* **Favourites**
* etc.
